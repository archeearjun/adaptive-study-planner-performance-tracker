Built a new portfolio project: **Adaptive Study Planner & Performance Tracker**

It is a Java 17 desktop application that combines:

- SM-2 spaced repetition
- priority-weighted daily planning
- Pomodoro-based focus sessions
- performance analytics
- a custom logistic regression model for recall prediction

What I wanted to build:

I did not want another static task manager. I wanted a planner that reacts to study behavior, updates revision timing, and explains why a topic is scheduled.

What makes it technically interesting:

- JavaFX UI with multiple product screens
- SQLite persistence with automatic schema bootstrapping and demo seeding
- adaptive replanning after session logging
- topic-level analytics and retention tracking
- custom logistic regression integrated into scheduling decisions

Tech stack:

`Java 17` `JavaFX` `SQLite` `Maven` `JUnit`
